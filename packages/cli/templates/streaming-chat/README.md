# {{name}}

Real-time streaming commerce chat scaffolded by `codespar init`.

## Setup

```bash
cp .env.example .env
# Fill in CODESPAR_API_KEY and OPENAI_API_KEY

npm install
npm run dev
# → http://localhost:3000
```

## What it does

Next.js 15 App Router + Vercel AI SDK + CodeSpar. Token-by-token streaming chat where tool calls happen automatically and appear live in the UI.
